# CCGrid 2027 Website

Static website for CCGrid 2027, hosted by the University of North Texas in the DFW Metroplex.

The publish directory is `ccgrid2026/`. The directory name is inherited from the previous edition, but the public content has been updated for CCGrid 2027.

To preview locally:

```bash
cd ccgrid2026
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
